## Description
Ramba Contact API is a simple address book API that helps organize your contacts and their details
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
#### Hosted link 
[Heroku](https://ramba-contact.herokuapp.com/)

- NestJs
- MongoDB
